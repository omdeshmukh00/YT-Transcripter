# 🎥 YT Transcripter

YT Transcripter is a modern Progressive Web App (PWA) that converts YouTube videos into clean, timestamped PDF transcripts with a single click.

Designed for users who regularly save, read, share, or archive YouTube content, the application extracts original video captions, preserves exact timestamps, and generates professional PDF documents with full Unicode support for English, Hindi, and Marathi.

---

## ✨ Features

* Paste any YouTube video URL
* Extract original YouTube captions
* Preserve exact timestamps
* Generate professional PDF transcripts
* Full Hindi, Marathi, and English Unicode support
* PDF preview before download
* Native mobile sharing support
* Local transcript history (No Database Required)
* Installable PWA for Android and iOS
* Offline-ready experience
* Mobile-first responsive design
* Fast and lightweight architecture

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/omdeshmukh00/YT-Transcripter.git
```

### Navigate into the Project

```bash
cd YT-Transcripter
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

in your browser.

## 🎯 Why YT Transcripter?

Most transcript tools either remove timestamps, require subscriptions, or fail to support Indian languages properly.

YT Transcripter focuses on:

* Accurate timestamp preservation
* Clean PDF exports
* Hindi and Marathi support
* Privacy-first local storage
* No database dependency
* One-click workflow
